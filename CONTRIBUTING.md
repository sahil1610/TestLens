# Contributing

Thanks for your interest in contributing to TestLens.

## Development setup

```bash
npm install
```

## Commands

- Build:

```bash
npm run build
```

- Unit + integration tests:

```bash
npm test
```

- Run the Playwright integration run directly (expected to fail intentionally and print TestLens output):

```bash
npm run test:pw
```

## Pull requests

- Keep changes **lightweight** and aligned with the project philosophy: rule-based, CLI-first, minimal setup.
- Add or update tests for behavior changes.
- Avoid introducing heavy dependencies unless clearly justified.
